If you are on Windows, here's how you can set up the inbuilt [Snipping Tool](https://support.microsoft.com/en-us/windows/use-snipping-tool-to-capture-screenshots-00246869-1843-655f-f220-97299b865f6b) to get your videos processed by YakShaver!

### Setting up Snipping Tool

1. In your OneDrive Recordings' folder, create a new Folder called `SnippingTool`, i.e. `OneDrive\Recordings\SnippingTool`
2. Press "Windows" and search for "Snipping Tool" to open the Snipping Tool app
3. Go to Snipping Tool | Settings | In the "Screen recording" section
4. Change the folder to the `SnippingTool` folder you just created
5. Include microphone input by default

   ![](/YakShaver/Docs/Snipping-Tool-Settings.png)\
   **Figure: Ensure that the folder location is set, and microphone is always on**

### Recording a video

1. Press "Windows + Shift + R" to open Snipping Tool in Recording mode
2. Select your recording window and record
3. When done, stop the recording and wait for YakShaver to process the video
